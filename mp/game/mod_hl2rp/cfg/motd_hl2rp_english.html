Welcome to Half-Life 2: Roleplay

“
This project aims to create an entertaining experience set in the Half-Life 2 world, focused on both the Citizen and Combine faction”

Important instructions

This mod makes an extensive use of server-wise, old-fashioned custom dialogs (e.g. menus).
They are required to interact with the very fundamental gameplay aspects of the mod.
For security concerns, they are disabled by default in the game.
Therefore, you ought to enable them. To do so, you may copy-paste the following command in chat:

";cl_showpluginmessages 1

Information for Half-Life 2: Deathmatch players

Following the above, there is a main menu which represents your entry point to the core gameplay.
In order to open it, you need to hold the reload key (default: R) for about two seconds.
